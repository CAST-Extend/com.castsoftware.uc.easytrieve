The following demo programs are included:


DEMODB2A.EZT - Sample program using Native SQL method.
DEMODB2A.COB - Generated COBOL of Native SQL.

DEMODB2B.EZT - Sample program using SQL File method.
DEMODB2B.COB - Generated COBOL of SQL File method.

DEMODB2C.EZT - Sample program using JOB INPUT SQL method.
DEMODB2C.COB - Generated COBOL of JOB INPUT SQL method.

DEMOESY1.EZT - Sample program using sequential files and one report.
DEMOESY1.COB - Generated COBOL of sequential files and one report.

DEMOESY2.EZT - A real production program with complex logic.
DEMOESY2.COB - Generated COBOL of real production program.

DEMOESY3.EZT - Sample program with Easytrieve macros and copybook option.
DEMOESY3.COB - Generated COBOL with COBOL COPY statements.
EZTCOPYA.EZT - Easytrieve Macro (layout of input/output file).
EZTCOPYA.CPY - COBOL copybook (layout of input/output file).

DEMOESY3.JCL - Sample JCL generated by the translator for DEMOESY3.
